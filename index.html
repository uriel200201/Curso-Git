    • $git revert --no-commit HEAD
  • 6to video
    • agregar ramas
    • $git branch (nombre de rama) crea una rama
    • $git checkout (nombre de rama) cambia de rama
    • $git checkout -b (nombre de rama) crea y te cambia de rama
    • $git checkout -d (nombre de rama) elimina la rama
    • $git merge (nombre de rama) fusiona la rama en la que estas con la que estas llamando
  • 7mo tema
    • como usar los tags
    • $git tag (nombre del tag) crea un tag en el commit HEAD
    • $git tag (nombre de rama) (hash del commit) Crea un tag para el commit mensionado
    • $git checkout (nombre del tag) Te cambia al commit con ese tag
    • $git tag muestra todos los tags que hay
    • $git tag -d (nombre de tag) Elimina el tag nombrado
    • $git tag -a (nombre de tag) Crea un tag anotado
  • 8vo tema
    • como usar el stash
    • $git stash crea un stash en la rama
    • $git stash list Lista todos los stash hay
    • $git stash apply stash@\{(numero de stash)\} aplica todos lo que estaba en el stash Si no se aclara aplicara el ultimo
    • $git stash drop stash@\{(numero de stash)\} Elimina el stash
    • $git stash save (mensaje que quieras poner) Hace un stash con un mensaje personalizado
    • $git stash pop stash@\{(numero de stash)\} Crea un tag anotado
